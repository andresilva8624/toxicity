
# Toxicity

![Badge](https://img.shields.io/badge/AndreGitHub-MIT-green.svg)

##Description

A pseudonymous social media application that allows an anonymous safe line of communication in your local area. A social platform that will allow the user to create an account privately and allow to express toxic posts/behavior in a local forum to share with many other users worldwide.
